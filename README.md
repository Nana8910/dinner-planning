# ツクリオリオ — 一週間こんだてプランナー

ツクリオキで届くおかずを1週間に振り分ける、スマホ対応の献立プランナー。

- 冷蔵のおかずは前半に、冷凍できるものは後半にまわして使い切る
- 当日あまったおかずを後日にまわす提案
- 自作のおかずも登録できる
- 週カレンダーで献立をひと目で確認

## 使い方

`index.html` をブラウザで開くだけ（ビルド不要）。データは各端末のブラウザに保存されます。

公開URL（GitHub Pages）から iPhone などのスマホでも開けます。

## ファイル

- `index.html` — そのまま動く本体（React + Babel をCDNから読み込み）
- `tsukuriorio.jsx` — 開発用のソース（Reactコンポーネント）
